# Algorithms and Data Structures

This module covers the fundamental algorithms and data structures every software engineer should understand to write efficient, scalable code.

## Sections

1. [Big O Notation](./01-big-o/README.md) - Understanding algorithm complexity and scalability
2. [Arrays and Objects Performance](./02-arrays-and-objects/README.md) - Time and space complexity of fundamental data structures
3. [Searching and Sorting](./02-searching-and-sorting/README.md) - Common algorithms for finding and organizing data
4. [Linked Lists, Stacks, and Queues](./03-linked-lists-stacks-queues/README.md) - Fundamental linear data structures
5. [Trees and Graphs](./04-trees-and-graphs/README.md) - Hierarchical and network data structures
6. [Recursion and Iterative Thinking](./05-recursion-and-iteration/README.md) - Approaches to solving problems
7. [Dynamic Programming (Basics)](./06-dynamic-programming/README.md) - Optimizing recursive solutions
8. [Problem Solving Patterns](./07-problem-solving-patterns/README.md) - Common algorithmic patterns and problem-solving strategies

## Learning path

Start with **Big O Notation** to build the foundation for understanding why certain algorithms are better than others. Then proceed through the sections in order, as later topics build on earlier concepts.
